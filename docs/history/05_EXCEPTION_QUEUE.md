## Closure rule

An exception closes only when:

- the original is recovered and integrity-registered;
- an evidence-bounded reconstruction is formally accepted;
- the unit is formally superseded with owner, authority, and effective date; or
- authoritative evidence establishes that the reference never became an artifact.
